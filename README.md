# MyDoctor

## Install

### One-Line Install for Linux

```bash
curl -fsSL https://raw.githubusercontent.com/lordvivaan/mydoctor-installer/main/install.sh | bash
```

### One-Line Install for Macos

```bash
curl -fsSL https://raw.githubusercontent.com/lordvivaan/mydoctor-installer/main/install.sh | bash
```

### One-Line Install for Windows

```bash
irm https://raw.githubusercontent.com/YOURUSERNAME/mydoctor-installer/main/install.ps1 | iex
```

## 📁 What Gets Installed

Ollama (main LLM runner)
LiteRT-LM (Google’s optimized on-device runtime)
Project folder: ~/mydoctor (Linux/macOS) or C:\Users\YourName\mydoctor (Windows)
Ready-to-use Modelfile
